# Polymarket Bot v10.0 — Dashboard

Telegram Mini App dashboard для Polymarket торгового бота.

**Ссылка:** https://petushokmaxorka-ai.github.io/polymarket-dashboard/

## Возможности
- AI Council — 6 персонажей (Карл Роув, Нейт Силвер, Сорос, Талеб, Бьюрри, Ren Tech Quant)
- Feedback Loop — Brier Score, динамические веса моделей
- Monte Carlo Risk Layer — n_paths=5000
- Checklist / Roadmap — СРОЧНО / ВАЖНО / ДОЛГОСРОЧНО
- VPN / Prometheus — статус серверов
- Симуляция — Risk Lab с графиком кривых капитала

## Использование как Telegram Mini App

Открывается через команду `/dashboard` в боте, либо по кнопке в меню.
